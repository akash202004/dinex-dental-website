# Dinex Dental Website

## Description

This project is a dentist website developed using HTML and CSS. It features mobile responsiveness and utilizes locomotive scroll for smooth navigation.

## Features

- Mobile-responsive design ensures compatibility across various devices.
- Locomotive scroll enhances the user experience with smooth scrolling.
- Clean and organized HTML/CSS code for easy maintenance and customization.

## Installation

1. Clone the repository to your local machine.
2. Open the `index.html` file in your preferred web browser.

## Usage

- Navigate through the website using scroll or click actions.
- Ensure compatibility across different screen sizes for optimal user experience.
